# gobot

## 使用方法

注意：目前只支持在linux上运行，如需其它平台运行，自行修改里面的资源地址即可

### 1.先赋予脚本的运行权限

```shell
chmod +x ./*.sh
```

### 2.登录qq，根据提示进行配置

```shell
./run.sh qq
```

### 2.登录bot，根据提示进行配置

```shell
./run.sh bot
```
